# Git-Github
O que é Git e Github?

Ambas ferramentas são utilizadas no dia a dia das pessoas que desenvolvem softwares, pelo simples motivo de ser bem simples para o gerenciamento do código fonte da aplicação e terem muitas outras funcionalidades para auxiliar na manutenção do código. 

## Git 

O Git é um sistema de versionamento de código distribuído, ele permite que várias pessoas trabalhem no mesmo conjunto de arquivos (repositório), ao mesmo tempo em que evita conflitos entre as alterações e cada pessoa tem a sua cópia do arquivo. 

## Instalando o GIT
* `Link do Git:` (https://git-scm.com/downloads)

## Github
O Github tem muita relação com o Git. GitHub é uma plataforma para gerenciar seu código e criar um ambiente de interação entre as pessoas que estão editando o código, utilizando o Git como sistema de controle.

## Instalando o Github
* `Link do Github:` (https://github.com)
